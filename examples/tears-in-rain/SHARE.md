# 发布包 · Tears in Rain

> 三个平台调性不同，别混用。文案保留了我的判断，拿去改成自己的口吻。

## 一句话（当作自我介绍用）
一扇永远下雨的窗。字写上去，雾会把它盖回来。

## 开场钩子（3 个候选，挑一个用在任何平台开头）

都可以改成自己的口吻，但基本语感保留：

**候选 A（推荐，未完成动作）**
> 深夜想给谁发一句"还记得吗"，打到一半删掉了。  
> 后来我把它打在了一扇下雨的窗上，看着雾把它盖掉。

**候选 B（画面代入）**
> 想象一扇永远在下雨的起雾玻璃。  
> 你用手指在上面划字，停下来，雾会从笔画的边缘一点一点爬回来，把字盖住。  
> ——我做了这个。

**候选 C（金句转折）**
> "All those moments will be lost in time, like tears in rain."  
> 那些时刻去哪里了？  
> 我做了个地方，让它们待一会儿。

## 演讲思路（大纲，给你当 prompter 用）

1. **开场钩子** → 从上面三个候选里选一个
2. **一眼**：镜头给到起雾玻璃 + 手指在上面写字的特写
3. **灵感从哪来**（见下面"讲灵感故事的原话"）
4. **想做成什么样**：一个能写点东西进去、看着它消失的地方。没有存档、没有分享、没有注册。只有这一次
5. **做的时候卡在哪**：一开始字是整片淡出，像 CSS 动画一样，假。后来改成雾气从笔画边缘一点一点爬回来——逐像素回填——才像玻璃真的在重新起雾
6. **现场演示**：打字 → 停手 → 雾吃字 → 台词浮现 → 面板调参数 → 拖图换背景
7. **收尾**：一句留白，不做 CTA

### 讲灵感故事的原话（讲的时候照着说即可）

> 前阵子又看了一遍《花样年华》。
> 
> 最后那场在吴哥窟——周慕云把脸贴着古迹的石缝，对着那个洞说话，说完拿泥和草把洞封起来。那场没有台词，只有配乐。我一直在想：他说了什么？
> 
> 差不多同一阵子，也重看了《银翼杀手》。Roy Batty 在雨里坐着，快死之前说了那句——"All those moments will be lost in time, like tears in rain."
> 
> 一个讲想说又说不出口的话怎么办。  
> 一个讲那些话最终会去哪里。  
> 
> 我就想，能不能做一个这样的地方。

## 功能清单

### 主要交互
- 键盘打字，支持中文 IME（全屏接管，不用输入框）
- 停止输入 4 秒自动触发"消散"
- 按 `Esc` 立刻触发消散
- 消散 + 台词浮出后，按任意键跳过、重置

### 右下角控制面板
- **雨量**：雨滴密度
- **玻璃模糊**：背景失焦程度
- **雾气亮度**：整层凝结雾的浓淡
- **折射**：雨水对背景的扭曲强度

### 背景自定义
- 把图片或视频直接**拖进浏览器**替换背景
- 也可用控制面板上传

### 内容
- 内置双语电影台词库：银翼杀手（Roy Batty）、王家卫（花样年华、阿飞正传等）
- 长台词自动用更小字号 + 更长停留

### 技术
- 纯静态页面，无构建，直接打开 index.html 即可
- WebGL 2 + 自定义 GLSL 片元着色器
- 雨滴物理改编自 Shadertoy [Heartfelt](https://www.shadertoy.com/view/ltffzl)（BigWings，CC BY-NC-SA 3.0）
- 文字 Canvas 2D 离屏渲染 → shader mask
- 消散：`reclaim` 变量驱动，雾从笔画边缘逐像素回填
- Web Audio API：雨声 + 消散弦乐
- 字体：Caveat / Ma Shan Zheng

## B 站

**标题**：深夜想发又没发出去的话，我做了个地方让它消失

**简介**：

深夜想给谁发一句"还记得吗"，打到一半删掉了。

后来做了这个——一扇永远在下雨的起雾玻璃。可以在上面打字，停下四秒，雾会从笔画边缘一点一点爬回来，把你写的东西悄悄盖掉。然后一句旧电影的台词浮出来，停一会儿，又消失。没有存档，没有记录。

纯静态页面，拖图片或视频进去能换背景。

- 不要用 Safari（WebGL 性能和颜色渲染会有差）

### 分辨率 / 帧率
- 至少 **1080p 60fps**。雨滴运动快，30fps 会糊
- B 站横版：电脑原生 1920×1080 直接录
- 小红书 3:4 / 抖音 9:16：后期裁剪，或用 iPhone 实机打开 `http://<电脑局域网IP>:8080`（先 `python3 -m http.server 8080`）用 iOS 屏幕录制

### 录屏软件
- **Mac**：OBS（免费精细）或 ScreenFlow（付费好剪）。QuickTime 也能用但不能调 60fps
- **Windows**：OBS 或 ShareX

### 音频
- 保留网页雨声。Mac 录系统音装 **BlackHole**（免费虚拟声卡），或 OBS 配"音频捕获（应用程序）"直接抓 Chrome
- 配音如有，建议后期单独录再叠
- 不要叠 BGM，雨声本身是作品的一部分

### 作品内参数（录之前先调好）
| 参数 | 建议值 | 理由 |
|---|---|---|
| 雨量 | 70% | 够忙但不抢字 |
| 玻璃模糊 | 60% | 背景有氛围但看不清 |
| 雾气亮度 | 75% | 雾回填视觉效果最明显 |
| 折射 | 50% | 雨打玻璃的扭曲感刚好 |

### 背景图
别用纯色。推荐三选一：
1. **《花样年华》酒店长廊**（2046 号房门口那张昏黄壁灯 + 虚焦）——情绪直接对上
2. **《银翼杀手 2049》雨夜霓虹街景**——赛博 + 雨，呼应片名来源
3. 自己找"虚焦昏黄路灯 + 深蓝夜色"的图

拖进浏览器即可。`picture/` 文件夹有 3 张备选。

### 环境
- Mac 打开"请勿打扰"，自动隐藏 dock 和 menu bar
- 关闭其他窗口、清理桌面
- 键盘声音可录进去（打字声是体验的一部分），但别让它盖过雨声

## 封面设计

### 小红书（3:4 竖版）
- **画面帧**：雾回填到 **60%** 的瞬间（字还能看清，但右侧已被吃掉一半）
- **玻璃上的字**：`还记得吗`（4 字，手写体，画面中偏上 1/3）
- **字的状态**：从右向左被吃到"得"字右边。"还记"完整，"得"开始模糊，"吗"几乎消失
- **背景**：拖入《花样年华》酒店长廊剧照（昏黄 + 虚焦）
- **叠加层**：
  - 左上角极小字：`做了一扇永远下雨的窗`（手写体白色 50% 透明，字号主字的 1/4）
  - 右下角极小字：`tears-in-rain`（衬线浅灰）
- **色调**：深蓝紫 + 暖橘黄 + 玻璃蒙白

### B 站（16:9 横版）
- **画面帧**：同上，雾回填到一半
- **玻璃上的字**：`如果多一张船票`（7 字，画面左半中央）
- **字的状态**：被吃到"一张"。右半边"船票"几乎看不见
- **背景**：深蓝紫过渡到深蓝 + 远处虚焦暖黄光点 + 老电影颗粒感滤镜
- **版式（左右分栏）**：
  - **左 60%**：字 + 玻璃画面
  - **右 40%**（深色区）：
    - 主标题：`做了一扇永远下雨的窗`（手写体大号白色）
    - 副标题：`一次关于遗忘的创作`（细体小号浅灰）
    - 右下：`tears-in-rain`（极小灰）

### 抖音（9:16 竖版 / 首帧）
抖音不做独立封面，首帧即封面。
- **首帧**：玻璃上 `对不起`（3 字，手写体，画面上 1/3 处——底部信息栏会挡）
- **字的状态**：从右侧被吃到"起"字一半
- **背景**：深蓝紫默认
- **不叠加任何标题**，让画面自己说话
- 视频到第 2–3 秒再淡入极小字（底部 1/5 处）：`一扇永远下雨的窗`（手写白色 70% 透明）

## Demo 录制清单

每个镜头要打的字都已写好。挑字原则：有情绪但不露骨、笔画美、长度合适、部分呼应台词库。

- [ ] **镜头 A · 雨夜空镜**（3 秒 × 2 take）
  - 画面：整屏起雾玻璃 + 雨滴斜划 + 背景虚焦灯光
  - 输入：**无**
  - 卖点：氛围 B-roll
  - 备注：先按 Esc 清屏

- [ ] **镜头 B · 打字笔画特写**（5–8 秒 × 3 take）
  - 画面：中上部近景
  - 输入：**`hello`** 或 **`晚安`**
  - 卖点：写字擦开雾气的视觉触感
  - 备注：打字声要清晰

- [ ] **镜头 C · 雾气回填（核心）**（15 秒 × 3 take）
  - 画面：全屏，字在画面中间偏上
  - 输入：**`如果多一张船票`**（7 字，呼应花样年华台词）
  - 流程：打完 → 停手不动 → 录全 4 秒等待 → 录全雾气回填 8–10 秒
  - 卖点：per-pixel fog reclaim，项目最独特的那一下
  - 备注：**从打完到字消失任何一帧都不能剪**

- [ ] **镜头 D · 台词浮现**（5 秒 × 3 take）
  - 画面：全屏
  - 操作：等系统自动抽台词
  - 卖点：情感收尾
  - 备注：抽到不合适按任意键跳过、重来。**理想抽到**"如果多一张船票"或 Roy Batty 那句

- [ ] **镜头 E · Esc 手动触发**（4 秒 × 2 take）
  - 画面：全屏
  - 输入：**`还记得吗`** → 立刻按 Esc
  - 卖点：主动消散的控制感

- [ ] **镜头 F · 控制面板调节**（6 秒 × 2 take）
  - 画面：主画面 + 右下角面板局部
  - 输入：先打 **`雨`** 留在画面，再拖**雾气亮度**滑块低→高
  - 卖点：字的可见度随雾气实时变化

- [ ] **镜头 G · 拖图换背景**（5 秒 × 2 take）
  - 画面：全屏
  - 输入：先打 **`你在哪`** 留在画面，再拖一张新图进去
  - 卖点："装进任何一个雨夜"

- [ ] **镜头 H · 旧版 vs 新版消散对比（仅 B 站）**（8 秒 × 1 take）
  - 画面：左右分屏。左边 `git checkout 5e5c443` 跑出整片淡出的旧版；右边当前 per-pixel reclaim 版
  - 输入：两边都打 `想你了`（3 字）
  - 卖点：技术卡点的可视化，展示创作者的打磨过程
