# 示例：Tears in Rain

这是用 `vibe-share` 跑在一个真实项目上产出的 `SHARE.md`。

**原项目**：Tears in Rain —— 一扇永远在下雨的起雾窗户，打字会被雾气悄悄盖掉，然后浮现一句旧电影的台词。纯静态网页，WebGL 2 + 自定义 GLSL。

**Skill 自扒的信息**：
- 项目 `README.md`（创作理念、体验流程、技术说明）
- 早期构思文档 `plan.md` / `idea1.md`
- `git log`（从 commit message 里扒到 "per-pixel fog reclaim for organic dissolve" 这个关键的创作卡点）
- 源码（`index.html`、`shader.js`、`fog.js`、`quotes.js` 等）
- `picture/` 目录下的备选背景图

**补问的信息**：受众和基调（文艺青年 / 创意编程社区），以及最想讲灵感故事还是技术故事。

看一眼 [SHARE.md](./SHARE.md) 就知道产出长什么样：
- 三个候选开场钩子
- 完整配音稿（抖音 25 秒 / 小红书 45 秒 / B 站 4 分钟）
- 《花样年华》和《银翼杀手》具体到秒的画面引用方案
- 每个镜头具体打什么字（镜头 C 打「如果多一张船票」）
- 三个平台各自的封面构图
- 拍摄前设置、Demo 清单、剪辑节奏
